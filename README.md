# Aufgabe 2: Multi-Umrechner App

## Features & Umrechnungen
- **Fläche in Fußballfelder:** Rechnet Quadratmeter in die standardisierte Größe von Fußballfeldern (ca. 7140 m²) um.
- **Alter in Minuten:** Berechnet anhand des eingegebenen Alters in Jahren die ungefähre Lebenszeit in Minuten.
- **Geld in Zeit (Große Beträge visualisieren):** Veranschaulicht Geldbeträge als Sekunden (1 Mio. = 11 Tage, 1 Mrd. = 31,7 Jahre). 
  - *Easter Egg:* Bei extrem hohen Beträgen (Billionen / Billiarden) gibt es humorvolle Anspielungen auf das Vermögen von Elon Musk :)
- **Länge in Cheeseburger (Bonus):** Rechnet Meter in die "amerikanische Maßeinheit" Cheeseburger um (ca. 9,5 cm Durchmesser).
- **Gigabyte in Bits (Bonus):** Rechnet Datengrößen exakt bis auf die Bit-Ebene um.

- **Dropdown-Menü:** Komfortable Auswahl der verschiedenen Modi.
- **Fehlerbehandlung:** Abfangen von leeren Eingaben oder ungültigen Zeichen via Toast-Nachrichten.
